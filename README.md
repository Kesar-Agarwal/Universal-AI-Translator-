# SpeakSync-AI
# 🎙️ SpeakSync AI: Universal voice-to-voice Translator

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://speaksync-ai.streamlit.app)
[![Python 3.9+](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/downloads/)
[![Gemini 1.5](https://img.shields.io/badge/AI-Gemini%201.5%20Flash-orange)](https://deepmind.google/technologies/gemini/)

**SpeakSync AI** is a professional-grade, real-time voice translation tool. By combining **OpenAI's Whisper** for speech recognition and **Google's Gemini 2.5 Flash** for intelligent translation, SpeakSync provides natural, context-aware communication across 20+ languages.

---

## 🌟 Key Features
- **🎙️ Real-Time Voice Input**: Record and process audio directly through the browser.
- **🧠 Contextual Translation**: Uses Gemini AI to ensure translations sound natural, not robotic.
- **🔊 Instant Voice Output**: High-quality text-to-speech (gTTS) to hear the result immediately.
- **🔍 Smart Detection**: Automatically identifies the speaker's language using Whisper.
- **📜 Translation History**: A built-in history log to track your previous conversations.
- **📱 Responsive UI**: Optimized for both mobile and desktop use with a clean "SpeakSync" interface.

---

## 🛠️ Tech Stack
- **Frontend**: Streamlit
- **Speech-to-Text**: OpenAI Whisper (Base Model)
- **Translation Engine**: Google Gemini 1.5 Flash
- **Text-to-Speech**: gTTS (Google Text-to-Speech)
- **Backend**: Python

## 📈 Project Impact
This project demonstrates the integration of multiple state-of-the-art AI models into a single user-friendly production environment. It solves the "None/False" state management issues common in Streamlit and implements a custom session-based history system for a better user experience.
